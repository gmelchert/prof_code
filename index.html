Python é uma linguagem de programação poderosa, simples e de fácil leitura. Ideal para desenvolvimento web, automação, ciência de dados e inteligência artificial.

C# é uma linguagem moderna e orientada a objetos desenvolvida pela Microsoft. Muito usada para aplicações desktop, web e jogos com Unity.

Java é uma linguagem robusta, multiplataforma e orientada a objetos. Amplamente utilizada em aplicações corporativas, web e Android.

R é uma linguagem voltada para análise estatística e visualização de dados. Muito usada em pesquisa, ciência de dados e bioinformática.
